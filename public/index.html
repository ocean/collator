Collator – fetches and organises

Collator is an API that collects and collates information from various sources and serves it as JSON, with varying amounts of caching.

Usage

Simply the hit the endpoints below. Data is plain JSON by default.

API endpoints

Note: greyed out endpoints are not fully implemented yet.

URL Data provided
/v1/statements/commerce Commerce media statements from the Commerce site.
/v1/statements/ministerials Ministerial media statements from the WA Govt Media Statements site - the Mines and Petroleum and Commerce and Industrial Relations portfolios.
/v1/statements/government All Government media statements from the Media Statements site.
/v1/tweets/dmirs Twitter updates from @DMIRS_WA and related accounts.
/v1/tweets/transport Twitter updates about Perth transport information, including @Transperth and @Perth_Traffic.
