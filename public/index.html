Collator – gets the news

Collator is an API that serves news feeds as JSON data.

Usage

Simply the hit the endpoints below to get data. Data is plain JSON by default. JSONP can be requested by adding a ?callback=yourCallbackFunctionNameHere query parameter to the URL.

Here are the API endpoints available:

URL Feed provided
/v1/commerce-news Commerce media statements from the Commerce site
/v1/ministerials Ministerial media statements from the WA Govt Media Statements site
